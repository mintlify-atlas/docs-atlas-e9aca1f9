## Required Skills

### Technical Skills

Most data science internships expect:

**Programming**:
- Python (NumPy, Pandas, Scikit-learn)
- SQL for data querying
- R (for some analytics roles)

**Machine Learning**:
- Supervised/unsupervised learning
- Deep learning frameworks (PyTorch, TensorFlow)
- Model evaluation and validation
- Feature engineering

**Math & Statistics**:
- Linear algebra
- Probability and statistics
- Calculus (for ML theory)
- Experimental design (A/B testing)

**Data Tools**:
- Jupyter notebooks
- Git version control
- Cloud platforms (AWS, GCP, Azure)
- Data visualization (Matplotlib, Tableau)

### Research Positions (🎓 Advanced Degree)

AI research internships often require or prefer:
- Master's or PhD in progress
- Published research papers
- Strong mathematical background
- Experience with novel ML architectures

## Interview Process

### Common Interview Stages

1. **Phone Screen** - 30-minute recruiter call
2. **Technical Screen** - Coding + ML fundamentals
3. **Take-Home Project** - Real-world data science case study
4. **On-site/Virtual** - Multiple rounds:
   - Coding interview (data structures, algorithms)
   - ML theory (model selection, evaluation, tradeoffs)
   - Case study (solve a business problem with data)
   - Behavioral interview

### Typical DS/ML Interview Topics

<Info>
  **Coding**: Implement algorithms, data manipulation, SQL queries
  
  **Statistics**: Hypothesis testing, distributions, sampling, bias-variance
  
  **ML Theory**: Overfitting, regularization, model selection, evaluation metrics
  
  **System Design**: Design a recommendation system, fraud detection pipeline
  
  **Case Study**: Given this data, how would you solve X business problem?
</Info>

## Application Tips

### Build Your Portfolio

1. **Kaggle Competitions** - Demonstrate ML skills with real datasets
2. **GitHub Projects** - Share your code and models
3. **Research Papers** - Publish at workshops or conferences
4. **Blog Posts** - Explain complex concepts or project walkthroughs
5. **Coursework** - Highlight relevant ML and stats classes

### Stand Out

- **Domain Expertise** - Combine ML with another field (biology, finance, etc.)
- **Research Experience** - Work with professors on ML projects
- **Open Source** - Contribute to ML libraries
- **Internships** - Previous data internships, even at smaller companies
- **Competitions** - Win or place highly in data science competitions
