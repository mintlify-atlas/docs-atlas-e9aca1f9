Thank you for your interest in contributing to the Pitt CSC and Simplify internship list!

## Who Can Contribute

Anyone can contribute to the internship list. Whether you've found an open internship posting or need to update an existing one, we welcome your contributions to help keep this database accurate and up-to-date.

## Why Contribute

By contributing, you help:
- Build the most comprehensive tech internship database
- Keep information accurate and current for fellow students
- Support the community in finding quality internship opportunities
- Ensure listings are properly categorized and easy to discover

## Contribution Workflow

<Steps>
  <Step title="Find an internship">
    Identify an internship posting that meets our requirements (valid category, US/Canada/remote, uses formal ATS)
  </Step>
  
  <Step title="Submit via GitHub issue">
    Create a new issue using either the "New Internship" or "Edit Internship" template
  </Step>
  
  <Step title="Fill in details">
    Complete all required fields including category, company, title, location, and application URL
  </Step>
  
  <Step title="Review and approval">
    A maintainer reviews your submission and adds an `approved` label
  </Step>
  
  <Step title="Automatic addition">
    GitHub Actions automatically adds the internship to listings.json and updates the README
  </Step>
</Steps>

## What Happens After Submission

Once your contribution is approved:
- The internship is automatically added to `listings.json`
- The appropriate README file is updated with proper categorization
- Special indicators are added (🔥 for FAANG+, 🎓 for advanced degrees, etc.)
- The issue is automatically closed
- Changes are committed with your attribution

<Note>
  Please make a new submission for each unique position, **even if they are for the same company**.
</Note>

## Questions

If you have any questions about contributing, please create a [miscellaneous issue](https://github.com/SimplifyJobs/Summer2026-Internships/issues/new/choose).
