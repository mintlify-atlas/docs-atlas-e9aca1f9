# Browse Tech Internships

Explore over 1,736 curated tech internship opportunities for Summer 2026. The list is maintained daily by the Pitt Computer Science Club and Simplify.

## How to Use the Listings

The main internship repository contains comprehensive tables organized by category. Each listing includes:

### Table Columns Explained

| Column | Description |
|--------|-------------|
| **Company** | Company name with link to their Simplify profile. Look for the 🔥 icon for FAANG+ companies |
| **Role** | Internship position title. The 🎓 icon indicates advanced degree requirements (Master's/PhD/MBA) |
| **Location** | Office locations where the internship is offered. May include multiple cities or remote options |
| **Application** | Direct links to apply and to the Simplify autofill tool |
| **Age** | How recently the posting was added (0d = today, 1d = yesterday, etc.) |

### Special Indicators

<Info>
  - 🔥 **FAANG+ company** - Top-tier tech companies like Google, Meta, Apple, Amazon, Microsoft
  - 🎓 **Advanced degree required** - Requires Master's, PhD, or MBA
  - 🛂 **Does NOT offer sponsorship** - No visa sponsorship available
  - 🇺🇸 **Requires U.S. Citizenship** - Must be a U.S. citizen
  - 🔒 **Application closed** - No longer accepting applications
</Info>

## Contributing New Listings

Found an internship that's not on the list? Help the community by submitting it!

1. Create a [New Internship issue](https://github.com/SimplifyJobs/Summer2026-Internships/issues/new/choose)
2. Fill in the company name, role, location, and application link
3. Select the appropriate category
4. Submit - it will be reviewed and added within 24 hours

See the full [contribution guidelines](https://github.com/SimplifyJobs/Summer2026-Internships/blob/dev/CONTRIBUTING.md) for more details.
